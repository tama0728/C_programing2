NAME
        tel - search, add, delete, list phone numbers

SYNOPSIS
        tel [OPTION]... [STRING]...

DESCRIPTION
        tel is a simple phone book program. It can search, add, delete, and list phone numbers.
        if no option and string are given, tel will display manual.
        if no option is given, tel will search for the given string in the phone book.

        -a
                add a phone number

        -d
                delete a phone number

        -l
                list all phone numbers

        All options operate independent and do not mix.
